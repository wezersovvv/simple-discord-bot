# simple-discord-bot


## Required Packages
- discord.py
- json

## How to use
- Create a bot on the discord developer portal
- Enable all intents
-  Copy the token
- Paste the token in the variable "token"
- Replace the necessary lines in the codeimage.png
- Run the file "main.py"
- Invite the bot to your server
- Enjoy

## Commands
- !ping
- !help
- !info
- !clear
- !kick
- !ban
- !unban
- !mute
- !unmute
- !slowmode
- !dm
