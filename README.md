# simple-discord-bot


## Required Packages
- discord.py
- json
- random
- configparser
- random
- os

## How to use
- Create a bot on the discord developer portal
- Enable all intents
- Copy the token
- Paste the token in the Token variable
- Replace the necessary lines in the main.py file
- Run the file "main.py"
- Invite the bot to your server
- Enjoy
